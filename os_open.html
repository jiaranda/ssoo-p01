os_open

NAME

os_open

SYNOPSIS

#include "os_API.h"

osFile *os_open(char *path, char mode);

DESCRIPTION

La función os_open se encarga de abrir un archivo y retornar un descriptor de este.

PARAMETERS

  • char *path: path en el que se encuentra el archivo.
  • char mode: indica si el archivo se abre en modo lescritura 'w' o o lectura 'r'

RETURN VALUE

os_open retorna un struct del tipo osFile que funciona como descriptor del archivo abierto.

ERROR HANDLING

  • Si hay un error al leer el disco, retornará un osFile inválido, con el mensaje "ERROR: os_open. Error while reading disk."
  • Si se intenta leer ('r') un archivo inexistente se retornará un osFile inválido, con el mensaje "Invalid operation".
  • Si se intenta escribir ('w') un archivo que ya existe o que su directorio no existe, se retornará un osFile inválido, con el mensaje "Invalid operation".
  • Si se intenta escribir ('w') y el bloque directorio está lleno, retornará un osFile inválido, con el mensaje "Error: os_open. The address block is full"

EXAMPLE

#include "os_API.h"

int main(int argc, char *argv[])
{
	// Se monta el disco
	os_mount("data/simdiskfilled.bin");
	
	// Se abre un archivo
	osFile *file = os_open("/memes/generic (123).png", 'w');
}
